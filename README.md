# chat-cli
ICC chat cli

### Prerequisites

- Installing mongodb [installation guide](chapter-1-chat/guidelines/installing-mongo.md)
- Connecting to a database

## Usage
## Installation
### Install the dependencies

```js
  $ npm install
```

### Create Symlink

```js
  $ npm install
```

### Commands
- List Messages (list or l)

  - $ chat-cli list

- Find a message (find or f)

  - $ chat-cli find [msg]

- Add Message (add or a)

  - $ chat-cli add
